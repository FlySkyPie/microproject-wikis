增強式學習 (Reinforcement Learning) 是機器學習中的一個領域，強調如何基於環境而行動，以取得最大化的預期利益。強化學習是除了監督學習和非監督學習之外的第三種基本的機器學習方法。與監督學習不同的是，強化學習不需要帶標籤的輸入輸出對，同時也無需對非最優解的精確地糾正。

增強式學習本質上非常的像電玩遊戲，它建模了幾個實體：

- 環境 (Environment)，通常以狀態 (State) 表示
- 互動實體 (Agent)
- 行為 (Action)
- 觀察 (Observe)
- 策略 (Policy)
- 獎勵 (Reward)

而增強式學習可以被描述為：

- Agent 透過 Observe 取得 State
- Agent 透過 Action 改變 State
- Policy 是影響 Agent 如何 Observe 和 Action 的演算法
- 給定除了 Policy 以外的其他條件，尋找最大化 Reward 的 Policy
