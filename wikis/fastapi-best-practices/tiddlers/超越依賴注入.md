
Pydantic是一個很棒的模式驗證器，但對於涉及呼叫資料庫或外部服務的複雜驗證，它還不夠。

FastAPI文件主要將依賴項展示為端點的依賴注入，但它們也非常適合請求驗證。

依賴項可用於根據資料庫約束驗證資料（例如，檢查電子郵件是否已存在、確保找到使用者等）。

```python
# dependencies.py
async def valid_post_id(post_id: UUID4) -> dict[str, Any]:
    post = await service.get_by_id(post_id)
    if not post:
        raise PostNotFound()

    return post

# router.py
@router.get("/posts/{post_id}", response_model=PostResponse)
async def get_post_by_id(post: dict[str, Any] = Depends(valid_post_id)):
    return post

@router.put("/posts/{post_id}", response_model=PostResponse)
async def update_post(
    update_data: PostUpdate,  
    post: dict[str, Any] = Depends(valid_post_id), 
):
    updated_post = await service.update(id=post["id"], data=update_data)
    return updated_post

@router.get("/posts/{post_id}/reviews", response_model=list[ReviewsResponse])
async def get_post_reviews(post: dict[str, Any] = Depends(valid_post_id)):
    post_reviews = await reviews_service.get_by_post_id(post["id"])
    return post_reviews
```

如果我們沒有將資料驗證放入依賴項中，我們將不得不為每個端點驗證`post_id`是否存在，並為每個端點編寫相同的測試。
