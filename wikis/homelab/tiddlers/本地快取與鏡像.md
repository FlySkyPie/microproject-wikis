[FlyPie](#FlyPie) 的 Homelab 搭建了許多快取與鏡像服務作為基礎設施，用於加速軟體開發與佈署，可透過以下方式使用這些服務：

## APT

```dockerfile
ARG APT_PROXY=http://192.168.0.138:3142
RUN echo "Acquire::http::Proxy \"$APT_PROXY\";" | tee /etc/apt/apt.conf.d/01proxy
```

APT 的實做似乎不包含 hostname，無法透過反向代理連線到鏡像服務，因此需要透過 IP 直連。

## Python

```dockerfile
RUN pdm config --local pypi.url "http://web.devpi.arachne/root/pypi/+simple"
RUN pdm config --local pypi.verify_ssl False
# or
RUN pip config set global.index-url "http://web.devpi.arachne/root/pypi/+simple"
RUN pip config set global.trusted-host web.devpi.arachne
```

## Node.js

```dockerfile
RUN corepack enable pnpm && pnpm config set registry http://web.verdaccio.arachne
```

## Hugging face

```shell
HF_ENDPOINT=http://huggingface.mirrors.solid.arachne
```
