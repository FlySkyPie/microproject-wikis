## 電池

筆電伺服器需要在供電條件長時間使用，必須限制電池充點上限，避免長時間處於滿載狀態。

- Framework 在 BIOS 即可設定電池充電上限。
- Lenovo 可透過工具 ([https://github.com/makifdb/lenopow](https://github.com/makifdb/lenopow)) 處理。

## 筆電螢幕問題

讓筆電不會因為螢幕蓋上而進入休眠模式，編輯 `/etc/systemd/logind.conf`：

```shell
HandleLidSwitch=ignore
```

執行指令重啟服務：

```shell
systemctl restart systemd-logind.service
```

但是上述設定會造成另外一個問題：螢幕蓋著還是繼續發光，於是還需要一個步驟，編輯 `/etc/default/grub`：

在 `GRUB_CMDLINE_LINUX_DEFAULT` 插入 `consoleblank=60`。

修改後記得運行：

```shell
sudo update-grub
```
