OpenAI API 是指一系列 HTTP API 的定義，用以供應用程式與類神經模型互動，目前已成為實質產業標準。目前最被廣泛使用的端點為：

```
GET /v1/chat/completions
```

OpenAI API 的發展路徑類似於 S3，但是嚴謹用詞仍須使用「OpenAI-compatible API」以跟「OpenAI 的 API/SDK」做概念上的區隔，因為語言使用尚未完成像 S3 那樣的轉變；OpenAI （公司）的大眾印象依然過於強烈。
