JSweet 是一個 Java 工具鏈，提供了 Java 與 Javascript 之間的抽象層。透過這些工具鏈允許開發者以 Java 撰寫軟體，並且轉換成 Javascript 後佈署成 Web 網頁應用程式。

## JSweet Candies

JSweet Candies 是一系列「Javascript 到 Java 的榜定」，例如：`candy-jquery` 允許開發者使用 Java 使用 jQuery 的 API。

## JSweet transpiler

JSweet transpiler 是一個「Java 到 Javascript 的轉譯器」。

## 與 TeaVM 的差異

JSweet 和 TeaVM 雖然都涉及「Java 與 Javascript 之間的轉換」，但是處理的方式與目的不同。TeaVM 是在瀏覽器上建立 JVM，使 Java 的軟體能夠在瀏覽器上運作。
